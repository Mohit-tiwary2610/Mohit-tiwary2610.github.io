# Mohit Tiwary - Professional Portfolio
 
A modern, animated portfolio website showcasing full-stack development, data science, and cloud/DevOps expertise.
 
## 🚀 Features
 
- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Dark/Light mode toggle with localStorage persistence
- ✅ Smooth scroll animations and transitions
- ✅ Interactive project cards with 3D tilt effect
- ✅ Animated typing effect in hero section
- ✅ Particle background animation
- ✅ Progress bars for skills visualization
- ✅ Contact and Hire Me forms with validation
- ✅ SEO optimized
- ✅ Fast loading performance

## 🌐 Deployment Instructions
 
### Deploy to Vercel (Recommended)
 
1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```
 
2. Navigate to project folder:
   ```bash
   cd mohit-portfolio
   ```
 
3. Deploy:
   ```bash
   vercel
   ```
 
4. Follow the prompts and your site will be live!
 
### Deploy to Netlify
 
#### Option 1: Drag & Drop
1. Visit [Netlify Drop](https://app.netlify.com/drop)
2. Drag the entire `mohit-portfolio` folder
3. Your site is live instantly!
 
#### Option 2: Git Integration
1. Create a new repository on GitHub
2. Push this folder to the repository
3. Go to [Netlify](https://netlify.com)
4. Click "New site from Git"
5. Connect your repository
6. Deploy!
 
### Deploy to GitHub Pages
 
1. Create a repository named `<your-username>.github.io`
2. Push all files to the repository
3. Go to Settings → Pages
4. Select main branch as source
5. Your site will be live at `https://<your-username>.github.io`
